# gitsave Wiki

这是`gitsave`的文档页面，使用者需要有一定的`git`基础。

Gitsave 是一个专为游戏存档设计的 Git 管理工具，提供简单的保存/回滚流程，并引入“路线”概念来管理分支结局。

项目基于libgit2开发，底层兼容git。

本 Wiki 专注于两类内容：

1. 完整的安装与使用教程（CLI / TUI），包括需要遵循的规则与注意事项。
2. 针对特定游戏的存档处理指南与注意点。
